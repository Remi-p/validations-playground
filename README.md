# Validations Playground

## Initialization

1. Install `yarn`
2. Inside this repository, launch the following command: `yarn`.

## Execute tests

Use the command `yarn test`. `yarn test --watch` let jest relaunch itself on files change.

## Prettify your code

Use the command `yarn prettier:write`.
